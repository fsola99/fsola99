# Configuration

```shell
# Identity Name
git config --global user.name "Federico Sola"

# Identity Email
git config --global user.email "fdrcsola@gmail.com"

# Company
git config --global user.company "Deloitte"

# University
git config --global user.education "UADE"

# Localisation
git config --global user.localisation "Buenos Aires, Argentina"
```
